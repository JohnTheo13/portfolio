Mobile Demo

It's a side project I've been working on the last couple of months. I started creating this project in order to play mime game with my friends. You create two teams and start playing, it gives different movies to every team and also counts the score. I used a variety of technologies for this Project: HTML, CSS, JavaScript (AngularJS), Python, PHP (in the first version). I even worked on Photoshop for creating the posters the backgrounds and the icons.

This is a project for a client to show case responsive design and CSS and JQuery animations.

The first request for HSBC´s project was a 360 viewer. The client´s request was to be able to rottate the object and also zooming in to reveal hidden spots links. To finish this project I used this repository and change it to make it a 360 viewer. When a hidden spot is clicked, a jssor slider opens whith animated images.
